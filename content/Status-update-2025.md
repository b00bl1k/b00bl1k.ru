## Про проекты

### uwan

Все изменения за текущий год связаны только с исправлением ошибок, выявленных
в процессе использования библиотеки в проекте метеостанции.

### tashwiki

Новый минималистичный проект для организации вики, генерируемой из .md файлов.
Основнаяфича это викиссылки с подстановкой символа `_` вместо пробела в путь
к файлу:

```md
Пример [[Название страницы]].
```

будет преобразовано в

```html
<p>Пример <a href="Название_страницы.html">Название страницы</a>.</p>
```

<https://b00bl1k.github.io/tashwiki/>

### @avtonomberbot

Не так много изменений, в основном добавление поддержки кодов новых регионов РФ,
которые появились в этом году: 105, 224, 323.

<a href="https://board.kolibrios.org/viewtopic.php?p=80620#p80620">форуме проекта</a>.

<https://git.kolibrios.org/b00bl1k/kterm>
